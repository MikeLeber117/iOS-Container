F2View Documentation

An F2AppView is designed to be a simple way to incorporate an an F2 App within your native iOS app. It works by getting the configuration you supply it and requesting the app manifest fromt the web. It then uses the manifest and configuration to generat an html container with the F2 app and appends additional Javascript in order for the app to communicate to the delegate of the F2AppView

Adding an F2AppView to your app

In order to present an F2 app view, follow these steps:

1. Create an instance of the F2AppView and set its frame.

example:
myF2AppView = [[F2AppView alloc]initWithFrame:myAppFrame];

2. Set its Configuration using setAppJSONConfig:

The configuration is in JSON format and must contain at least the manifestUrl and the app ID. a configuration should look something like this:

[{
"appId": "com_f2_examples_javascript_watchlist",
"manifestUrl": "http://www.openf2.org/Examples/Apps",
"name": "Watchlist"
}]

more info on configuration objects http://docs.openf2.org/sdk/classes/F2.AppConfig.html

3. Configure the view using any of the optional properties

  • delegate
  • shouldOpenLinksExternally
  • scrollable
  • userScalable
  • additionalCss
  • scale

4. Register any events you want to listen to.

When using registerEvent:key:dataValueGetter: you set up a way for the app to access its delegate and call its F2View:messageRecieved:withKey: method

More info on events http://docs.openf2.org/sdk/classes/F2.Constants.Events.html

example:

[myF2AppView registerEvent:@"F2.Constants.Events.APP_SYMBOL_CHANGE" key:@“symbolchange” dataValueGetter:@"data.symbol"];

5. Load the app:

this should be done only after all the previous configuration is completed

loadApp: Call this method when your configuration is complete and you wish to load/reload the f2 app in the view

Sending Javascript commands:

sendJavaScript: Use this method to send any Javascript functions to the app.
